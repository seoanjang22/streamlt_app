# 📺 YouTube 댓글 분석기

YouTube 영상 URL을 입력하면 댓글을 수집하고 AI 기반으로 분석하는 Streamlit 웹 애플리케이션입니다.

## 주요 기능

- YouTube 영상 정보 분석
  - 제목
  - 채널명
  - 조회수
  - 좋아요 수
  - 댓글 수
  - 썸네일

- 댓글 분석
  - 최대 5,000개 댓글 수집
  - 작성자
  - 댓글 내용
  - 좋아요 수
  - 작성일

- AI 감성 분석
  - 긍정
  - 중립
  - 부정

- 시각화
  - 감성 비율 Pie Chart
  - 감성별 Bar Chart
  - 댓글 길이 분석
  - 인기 댓글 TOP10
  - 키워드 빈도 분석
  - 한국어 WordCloud

- 데이터 다운로드
  - CSV 파일 저장
